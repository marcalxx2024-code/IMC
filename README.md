# Calculadora IMC

Projeto simples contendo a página HTML para cálculo do Índice de Massa Corporal (IMC).

**Descrição:**
- Página: calculadora-imc.html — interface para inserir peso e altura e calcular o IMC.

**Requisitos:**
- Navegador moderno (Chrome, Edge, Firefox, etc.). Nenhuma dependência extra.

**Como usar:**
1. Abra o arquivo [projeto imc/calculadora-imc.html](projeto imc/calculadora-imc.html) no navegador.
2. Insira seu peso e altura e clique em calcular.
